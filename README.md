# ![Divide-and-Conquer Sorts](./assets/hero.png)

## Overview

This lesson covers two sorting algorithms that use the "divide-and-conquer" method: merge sort and quick sort.

### Prerequisites

- Big O Notation
- Recursion
- Intro to Sorting

### Learning Objectives

By the end of this module, learners will be able to:

- Describe how to use merge sort and quick sort to sort data.
- Explain the space and time complexities of merge sort and quick sort.
- Identify when to use merge sort or quick sort in a given scenario.

## Content

There are two components to this module:

1. Lesson: [Divide-and-Conquer Sorting Algorithms](./lesson/README.md)
2. Exercise: [Implementing Merge Sort & Quick Sort](https://github.com/ga-curriculum/divide-and-conquer-sorting-algorithms-starter-code)

## Solution Code 

- Exercise Solution: [Merge Sort](./solution/merge-sort.md)
- Exercise Solution: [Quick Sort](./solution/quick-sort.md)

### Time to Complete

2.5 hours total:

- 0.5 hour - Divide-and-Conquer Sorting Algorithms Lesson
- 2 hours - Implementing Merge Sort and Implementing Quick Sort Exercises
